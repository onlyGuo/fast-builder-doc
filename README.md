## 什么是FastBuilder？
"FastBuilder" 基于java17 和 vue3 快速开发框架，服务端99.99%以上的功能只需要配置数据模型编写写纯粹的业务脚本就能完成开发，另外剩下的0.01%也可以通过FastBuilder内置的API，一两行代码就可以搞定！不需要发布部署，不需要重启来实现研发团队的快速编码，提测以及回归。实现了服务端研发效率600%-900%的提升，人力成本减少了5倍。

2. 开发者全栈化，秩序一名前端程序员就可以独立完成整个项目的开发工作。
